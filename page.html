Terminologies

C++ Program Basics

  • Programming instructions must be written according to syntax rules.

Tokens,Identifiers,Keywords

Tokens

      • These are called reserved words or keywords.

      • Reserved words are always lowercase.

                                        int,float,double,char,void,return,etc

      • keywords cannot be used as an identifier.

Conditional Statements

What are condtional statements

  • Condtional statements are also known as decision making statements,they help to take decision in a program on the basis of logical conditions.

Different conditional statements in C++

  • if 
                        syntax :- 
                        if(condition){
                            //Block of code;
                        }
                        Here,if the condition is true ,only then the block of code within the brackets is executed.
  • if-else statement 
                            syntax :- 
                        if(condition){
                            //Block of code;
                        }
                        else{
                            //Block of code;
                        }
                        Here,if the condition is true ,only then the block of code within the brackets is executed otherwise the block of code in the else part is executed.
  • if-else if statement 
                            syntax :- 
                         if(condition 1){
                            //Block of code;
                        }
                        else if(condition 2){
                            //Block of code;
                        }
                        else if(condition 3){
                            //Block of code;
                        }
                        else if(condition 4){
                            //Block of code;
                        }
                        else{
                            //Block of code;
                        }
                        Here,if condition 1 is executed out of all the give blocks then only if condition is executed,
                        similarly if condition 1 is fail then it check to condtion 2 if it is corrrect it executes,if condition 2 is wrong goes to next condition to execute.
                        If all the condtions are wrong then the else part is executed.

  • Switch Case statement
    In Switch case, an expression is evaluated ,and the result of the expression is used to execute a set of statements associated to that value.They are used in place of longer if statements if we are constantly making comparsions to integral values. 
                            syntax :- 
                        switch(n){

                            break;

Input/Output in C++

Input Output in C++

  • In C++,all input and output are performed on a sequence of bytes called stream.
  • If bytes follow from a device like a keybord, a disk drive,or a networkconnection etc.to main memory ,this is called input operation.
  • If bytes flow from main memory to a device like a display screen, a printer, a disk drive, or a network connection, etc., this is called output operation.
  • iostream library defines the clin, cout ,cerr objects for the standard input stream, standard output stream and standard error stream.
  • fstream declare the services for user-controlled flie processing.

The different operators used for Input and Output.

    Input in C++

  • In C++,standard input is done using the cin operator of the istream class.
  • Used along with the extration operator(>>).
  • Automatically type casts the inputted value to that of the variable.
  • For strings,it takes the input to the first whitespace character.
  • For inputting strings with whitespace character,there is getline() function.

Output on C++

  • In C++,standard output is done using the count operator of the ostream class.
  • It is used to print to the console or other devices.
  • Used along with the insertion operator(<<).
  • Does not insert a new line character at the end of the output.

Manipulators in C++

  • Manipulators belong to the iomanip library.
  • They are functions that help to modify what we are writing to the output stream.
  • They help in formatting the output.
  • some commonly used manipulators are :-
    1. endl : Used to enter a new line.
    2. setw(val) : It is used to set the width of the all field for the output operations.
    3. setfill(c) : used to fill the (leading) space of the width with character.
    4. left : aligns the output to the left of the width.
    5. right : aligns the output to the right of the width.
